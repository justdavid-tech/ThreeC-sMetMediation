# Three C's Met - Mediation Institute

A premium, professional web platform for the Mediation Institute, specializing in Conflict Resolution, Strategic Negotiation, and Psychological Counseling. The website is built with a focus on high-end aesthetics, accessibility, and modern user experience.

##  Key Features

- **Core Pillars Framework**: An integrated system of four primary services:
  - **Mediation**: Confidential and voluntary dispute resolution.
  - **Negotiation**: Strategic engagement and professional advisory.
  - **Counseling**: Psychological preparation for lasting peace.
  - **Conciliation**: Expert-led settlement facilitation.
- **Premium Design System**: 
  - Responsive `100vh` Hero sections with interactive background layers.
  - Custom SVG decoration and high-quality React icon integration (`fa6`, `md`, `hi2`).
  - Sophisticated typography using Georgia serif for headers and modern sans-serif for body text.
- **Interactive Portfolio**: A hub for all services allowing users to explore the full architecture of resolution.
- **Social Impact Tracking**: Integrated stats and impact counters for certified peer mediators and outreach programs.
- **Dynamic UX**: Micro-animations, diagonal background lines, and interactive service cards.

## 🛠️ Technology Stack

- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS
- **Iconography**: React Icons (`fa6`, `md`, `hi2`, `gi`)
- **Routing**: React Router DOM
- **Animations**: Standard CSS transitions and dynamic JS-driven counters.

## Project Structure

- `/src/pages`: Main overview pages (Home, About, Services, Training, Contact).
- `/src/services`: Detailed service-specific pages for the core pillars.
- `/src/components`: Reusable UI elements (Hero, Pillars, Testimonials, etc.).
- `/src/assets`: Image and media resources.

##  Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

##  Philosophy

The Institute operates on the **"Three C's Met"** principle:
- **Conflict**: Acknowledged and addressed.
- **Communication**: Enabled and strategic.
- **Counseling**: Prepared and sustainable.
